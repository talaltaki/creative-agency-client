# Creative Agency
### A Full Stack Web Application for a Company which provides different Web Services
### [Live Site](https://creative-agency-webapp.web.app/)

### Features
A User can -
* Login with google
* Order services
* View order status
* Give feedback

An Admin can -
* Add new services
* View all orders
* Change order status
* Make new admins

### Technologies used
* JavaScript
* React
* React Router
* Bootstrap
* HTML
* CSS
* NodeJs
* MongoDB
* Firebase
* Heroku
